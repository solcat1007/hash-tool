### 哈希计算工具 文本与文件哈希计算工具，支持五种主流算法。 **主要功能** - 文本哈希：输入文本即时计算，支持单选或全部五种算法 - 文件哈希：拖拽或上传任意文件，计算文件哈希值 - 算法支持：MD5 / SHA-1 / SHA-256 / SHA-384 / SHA-512 - 大小写切换、一键复制结果 - MD5 纯 JS 实现，SHA 系列基于 Web Crypto API **注意事项** - 文件哈希计算大文件时可能耗时 - SHA-512 安全性最高但哈希值最长
